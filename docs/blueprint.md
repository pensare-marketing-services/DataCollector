# **App Name**: GatherWise

## Core Features:

- User Authentication and Profile Creation: Implement a login form that captures photo, phone number, age, Mandalam, Mekhala, and Unit information. Validate the data upon submission.
- Data Display and Confirmation: Display the collected user information in a standard format for review. Include a declaration statement and an 'Accept' button.
- Data Export: Provide an option to download the user data. File type: CSV.
- Sharing Functionality: Enable sharing the user information through various apps like WhatsApp and email.
- Dashboard Filters: Implement filters for name, age, Mandalam, Mekhala, and unit in the admin dashboard.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey trust and reliability.
- Background color: Light gray (#F5F5F5), nearly white, for a clean and neutral backdrop.
- Accent color: Warm orange (#FFB347) for the 'Accept' button and other call-to-action elements, to draw the user's attention.
- Font: 'PT Sans', a humanist sans-serif that combines a modern look with some warmth; for headlines and body text.
- Use a clean and responsive layout to ensure a seamless user experience across devices. The data display should be well-organized and easy to read.
- Employ clear and intuitive icons for actions like 'download' and 'share'. Icons should be consistent and align with the app's overall style.
